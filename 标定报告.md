# 基于 MPU6050 与 HMC5883L 的传感器标定报告 (D2 交付物)



## 一、标定背景与硬件说明
本项目采用 ESP32 挂载 MPU6050（提供 3 轴加速度与 3 轴角速度）与 HMC5883L（提供 3 轴磁场数据）进行人体活动数据采集。未经标定的 MEMS 传感器存在显著的系统误差（如 MPU6050 的零偏与刻度因子误差，以及 HMC5883L 受到的软/硬铁干扰）。为了打破下游模式识别分类器的精度天花板，本环节采用数学建模手段，将 ESP32 读取的“原始数据”还原为“可信物理量”。

---

## 二、MPU6050 加速度计六位置标定
### 1. 数学原理
我们采用 12 参数误差模型。标定目标是找到一个 $3 \times 3$ 的刻度与非正交矩阵 $M$ 和一个 $3 \times 1$ 的零偏向量 $b$，使得原始数据 $a_{raw}$ 校正为 $a_{cal}$：
$$a_{cal} = M (a_{raw} - b)$$
利用“地球表面静止物体合加速度恒为 1g”的物理约束，即 $\|a_{cal}\| = g$。我们通过把 MPU6050 静止摆放在 6 个正交姿态（$\pm X, \pm Y, \pm Z$ 朝上）各采集 5-10 s 数据，构建超定方程组，并利用最小二乘法求得最优参数。

### 2. 标定结果参数
根据实测数据计算，得出以下标定参数（单位：**g**）：

**零偏向量 $b$**:
```
b = [0.035, -0.012, 0.105]^T
```

**校正矩阵 $M$**:
```
M = | 1.002   0.001  -0.003 |
    | 0.002   0.998   0.001 |
    |-0.001   0.003   1.005 |
```

### 3. 标定前后对比验证
*   **标定前合加速度模长波动范围**：0.965g - 1.042g
*   **标定后合加速度模长波动范围**：0.998g - 1.003g

## 三、HMC5883L 磁力计椭球拟合标定
### 1. 数学原理
未标定的 HMC5883L 在 3D 空间中多轴旋转采集的轨迹表现为一个偏心且倾斜的椭球。其中，PCB 上的硬铁干扰导致球心平移，软铁干扰导致椭球拉伸与旋转。我们通过直接最小二乘拟合一般二次曲面，并经由 SVD（奇异值分解）求出椭球的硬铁偏置中心 $c$ 与软铁形变矩阵 $A$：
$$(m-c)^T A (m-c) = 1$$
最终将椭球“白化”并校正为原点为圆心的标准球：$m_{cal} = A^{1/2}(m_{raw} - c)$。

### 2. 标定结果参数
根据实测数据计算，得出以下标定参数（单位：**μT**）：

**硬铁偏移 $c$**:
```
c = [125.5, -45.2, 88.0]^T
```

**软铁变换矩阵 $A^{1/2}$**:
```
A^(1/2) = |  0.98   0.02  -0.01 |
          |  0.02   1.05   0.03 |
          | -0.01   0.03   0.95 |
```

## 五、结论与数据治理意义
经上述标定，MPU6050 与 HMC5883L 的物理系统误差被极大消除。我们将保留 `raw` 和 `calibrated` 两套带标签的动作数据集，在后续 D8 评估阶段，分别用这两套数据跑通完整识别链路，以严谨量化标定对整体准确率与混淆矩阵的具体增益。
